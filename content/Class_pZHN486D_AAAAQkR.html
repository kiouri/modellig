Версия формата public string 4.04

Принимает значение: 4.04

Тип информации public string

Атрибут принимает значения из справочника «Тип информации».

Перечень значений:

- ЕГРЮЛ_ОТКР_СВЕД – при передаче информации с открытыми сведениями (отсутствуют сведения о документах, удостоверяющих личность, и адресах физических лиц);

- ЕГРЮЛ_ЗАКР_СВЕД – при передаче информации с закрытыми сведениями (присутствуют сведения о документах, удостоверяющих личность, и адресах физических лиц).
Версия передающей программы public string Требования к атрибуту не определены. Рекомендуемые требования к значениям атрибута:

Атрибут должен содержать сведения о версии передающей программы.

Значение атрибута может содержать только следующие символы:

- заглавные (прописные) и строчные буквы русского алфавита – [A, Я] и [а, я];

- заглавные (прописные) и строчные буквы латинского алфавита – [A,Z] и [a, z];

- арабские цифры – [0, 9];

- французские кавычки («ёлочки»);

- знак «номер»;

- знак «дефис»;

- знак «тире»;

- словоразделитель – одиночный символ «пробел».

- знак «запятая»;

- знак «точка».

Данные требования могут быть усилены дополнительными правилами.
